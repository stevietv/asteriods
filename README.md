
# Asteroids — Boot.dev project

This repository contains the Asteroids project implemented as part of the [Boot.dev](https://boot.dev). curriculum  It uses Pygame for rendering and input.

### Quick start
- Install dependencies: `pip install pygame`
- Run the game: `python main.py`

### Notes
- This codebase was built following the Boot.dev guided project. See https://boot.dev for the original course and tutorials.

### Files of interest
- `main.py` — entry point that launches the game
- `player.py`, `asteroid.py`, `shot.py` — core game objects
- `constants.py` - definition of game constants
